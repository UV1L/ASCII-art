ASCII Art 
Версия 1.6

Автор: Васянович Антон (antonvasyanovich1@gmail.com)

Описание
	Скрипт позволяет преобразовать любое изображение в ASCII-арт (картинка нарисована символами таблицы ASCII)

Требования
	Python версии не ниже 3.6

Состав
	Консольная версия: Drawer.py
			   Gui.py
			   Main.py
	                   RGB_Converter
			   test_main.py
			   requirements.txt
		           test.png
			   test.txt

Запуск
	Запустить скрипт любым возможным способом. Можно через терминал (запускать Main.py) как без ключей (чтобы воспользоваться gui), так и с ключом -f или --file, чтобы передать картинку напрямую. При необходимости воспользоваться справкой по ключу -h либо --help.
